Тип селекторів

  1. Селектор тегу - надає стилі всім елементам з однаковим тегом.Має найменший приорітет.
  2. Селектор за класом - надає стилі для елементів для яких вказаний атрибут class.
  3. Селектор за атрибутом - дозволяє звернутись до елементу і надати стилі за його атрибутом. Має однакові приорітети з селектором класу
  4. ID селектори - самий пріоритетний тип селекторів, стилі вказані з цим селектором застосовуються першочергово.
  5. За вкладеністю - встановлює стилі для конкретного елементу або елементу який знаходиться в середині іншого

Command list

  • git status-показує зміни
  • git add .- додає файли до майбутнього коміту
  • git commit -m "title" - фіксує додані зміни
  • git branch- показує кількість гілок комітів
  • git checkout - перехід між гілками
  • git stash- відмінити зміни
  • git checkout -b "назва гілки"створення додаткової гілки
